# ネットワーク設定標準

## 開発環境の外部接続設定

### 固定設定
- **ホスト**: 0.0.0.0 (外部からのアクセスを許可)
- **ポート**: 3001 (スマホ接続用)
- **アクセスURL**: http://192.168.1.7:3001

### package.json スクリプト
```json
{
  "scripts": {
    "dev": "next dev -p 3001 -H 0.0.0.0",
    "dev:offline": "next dev --hostname 0.0.0.0 --port 3000",
    "dev:local": "next dev --hostname 0.0.0.0 --port 3000"
  }
}
```

### 起動手順
1. `npm run dev` で開発サーバー起動
2. 外部接続確認: `http://192.168.1.7:3001`
3. スマホからアクセステスト

### トラブルシューティング
- 接続できない場合: ファイアウォール確認
- ポート競合: `netstat -an | findstr :3001`
- IP確認: `ipconfig`

### 重要事項
- 常に外部接続可能な状態を維持
- 修正後は必ず外部アクセスを確認
- スマホでの実機テストを最優先
